## 🗂️ Project Structure Reference

```
lib/
├── main.dart                              # App entry point
├── theme/
│   └── app_theme.dart                    # Complete theme system
├── widgets/
│   └── custom_widgets.dart               # Reusable components
├── models/
│   ├── location_model.dart               # Location data model
│   └── route_model.dart                  # Route data model
├── services/
│   ├── location_service.dart             # Location management
│   ├── navigation_service.dart           # Route calculations
│   └── voice_service.dart                # Voice guidance
├── providers/
│   └── navigation_provider.dart          # State management
└── screens/
    ├── home_screen.dart                  # Home page
    ├── navigation_screen.dart            # Navigation page
    ├── location_details_screen.dart      # Details page
    ├── locations_list_screen.dart        # List page
    └── settings_screen.dart              # Settings page
```
